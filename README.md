# Eduka FrontEnd Internship Test

## About the application
Simple website for doing an exam. Build with VueJS.

### To compile and run
```
npm run serve
```

### To turn on mock server for quiz database
```
json-server db/questions.json
```

### To turn on mock server for user database
```
json-server db/user.json --port 3004
```

note: before running all the command above change directory to vueapp first
```
cd vueapp
```
